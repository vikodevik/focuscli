## üöÄ How to Use focuscli

After installation, you open focuscli in your terminal or command prompt. Here are the main things you can do:

### Managing Tasks

- **Add a task:**  
  Type `focuscli add "Task description"` and press Enter.  
  Example:  
  ```
  focuscli add "Write project report"
  ```
  
- **List tasks:**  
  Type `focuscli list` to see all your current tasks with numbers.  
  
- **Mark a task as done:**  
  Type `focuscli done [task number]` replacing `[task number]` with the number from the list.  
  Example:  
  ```
  focuscli done 2
  ```
  
- **Delete a task:**  
  Type `focuscli delete [task number]` to remove an unwanted task.

### Using Pomodoro Timer

The Pomodoro technique divides work into 25-minute focus intervals followed by 5-minute breaks.

- **Start a Pomodoro:**  
  Type `focuscli pomodoro start`. It will count down 25 minutes.  

- **Pause the timer:**  
  Type `focuscli pomodoro pause`.  

- **Stop the timer:**  
  Type `focuscli pomodoro stop`.

- **View timer status:**  
  Type `focuscli pomodoro status`.

By using this timer, you can work in bursts to maintain focus and avoid burnout.
